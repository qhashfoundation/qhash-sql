# qhash-sql

**qhash-sql** is an experimental SQL-like service by **Qhash Labs**.

## Status
- **Experimental / Proof-of-Concept**
- APIs, behavior, and data model may change at any time.

## Data retention
- **Data is not persisted.**
- Stored data may be **deleted automatically every 24 hours**, or **at the discretion of Qhash Labs**, without notice.
